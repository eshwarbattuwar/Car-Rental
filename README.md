## ğŸš€ Getting Started

Follow these steps to run the project locally:

1. Clone the repo:

```bash
git clone https://github.com/eshwarbattuwar/Car-Rental.git
```

2. Navigate to the project folder:

```bash
cd Car-Rental
```

3. Install dependencies:

```bash

```

```bash
npm start
```
